Projects

During my student career I have always had a deep interesting in trying to apply some of the concepts I learned into actual products and projects.

Student room booking - For the Internet Bases Systems Architecture Course Final Project I built a Ruby on Rails app that connects to Postgre SQL database, as one part of a project to manage student booking of rooms to study on.
Salary Calculator - While working with Landing.jobs I built a Ruby on Rails app that allows you to calculate your net salary from your gross income and allowances for the Portuguese and UK taxing system.
Geometry Friends - While interning at INESC I developed the circle agent for the game Geometry Friends, using Rapidly Exploring Random Trees.
Concurrent Web Server in C - Built a multi-threaded concurrent web server in C for the course Systems Programming of IST
Dynamic Bayesian Network Learning - Built a project for value prediction using Dynamic Bayesian Networks in Java for the Object Oriented Programming course.
Smart Kettle - During an HackerSchool event LX Reactor 2014, built a smart kettle together with four other students. We bought and modified a cheap kettle and implemented a temperature sensor and controlled the water temperature in order to be able to serve tea at just the right temperature.
